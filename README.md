# Amazon-Prime-Video-Dashboard

## Project Overview

This project is an interactive Amazon Prime Video Dashboard built using
Power BI to analyze streaming platform content data. The dashboard
provides insights into movies and TV shows available on the platform,
including genre distribution, content trends, ratings, release years,
and country-wise availability.

## Project Objectives

-   Analyze total content available on the platform
-   Compare Movies vs TV Shows distribution
-   Identify top genres and categories
-   Analyze content release trends over the years
-   Examine country-wise content availability
-   Evaluate ratings distribution

## Tools & Technologies Used

-   Power BI Desktop (.pbix)
-   Excel / CSV Dataset
-   Power Query
-   DAX (Data Analysis Expressions)

## Dataset Description

The dataset contains: - Show ID - Title - Type (Movie / TV Show) -
Director - Cast - Country - Date Added - Release Year - Rating -
Duration - Genre - Description

## Key KPIs

-   Total Titles
-   Total Movies
-   Total TV Shows
-   Content Added Per Year
-   Titles by Genre
-   Titles by Country
-   Ratings Distribution


## Skills Demonstrated

-   Data Cleaning & Transformation
-   Data Modeling
-   DAX Measures Creation
-   KPI Development
-   Interactive Dashboard Design
-   Business Insight Generation

## Conclusion

This project demonstrates practical application of data analytics in the
entertainment industry. Using Power BI, Power Query, and DAX, raw
streaming data is transformed into meaningful business insights through
interactive dashboards.
